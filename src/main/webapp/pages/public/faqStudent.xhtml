Principais Dúvidas de Acesso

 Os estudantes também necessitam estar cadastrados na plataforma?

Não. O acesso dos estudantes será apenas durante o jogo e não necesita de cadastro.

 Como faço para acessar a plataforma?

O acesso dos estudantes deve ser feito utilizando um smartphone através da leitura de tags NFC ou QR-codes disponibilizados pelos professores.

 Como iniciar uma missão?

A missão será iniciada ao efetuar a leitura da primeira tags NFC ou QR-code disponibilizada pelo professor.

 Como faço para criar minhas próprias missões?

Apenas professores cadastrados na plataforma podem criar suas prórpias missões.
